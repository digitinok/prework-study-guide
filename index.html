HTML

  • provides structure of a website
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS - Cascading Style Sheets
  • determines the styles of a webpage, making a website visually appealing
  • defines the colors, aesthetics, visual layout, and organization
  • three ways to style a webpage:
    • inline styles: style a particular element directly in the HTML, quick & permanent, but less flexible as individual element only styled
    • internal style sheets: embedded CSS rules in the HTML file using a <style> element in the <head> element, only apply to that page
    • external style sheets: most common way to style a webpage, keep all their CSS rules in a separate file, link the file using a <link> element placed in the HTML's <head>
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
